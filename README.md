# Sélection Fournisseurs — projet Vercel

## Lancer en local
1. Installer Vercel CLI
2. Copier `.env.example` vers `.env.local`
3. Renseigner le token ClickUp
4. Lancer `vercel dev`

## Variables d'environnement
- `CLICKUP_TOKEN`
- `CLICKUP_LIST_NEWS`
- `CLICKUP_LIST_CHALLENGES`

## Déploiement Vercel
1. Importer le dépôt dans Vercel
2. Ajouter les variables d'environnement dans Project Settings
3. Déployer
